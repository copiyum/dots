# macOS Rice

macOS rice wee

## Install

```sh
scripts/rice-apply --install-deps white
```

## Apply

```sh
scripts/rice-apply white
scripts/rice-apply black
```

## Default Keystrokes

| Key | Action |
| --- | --- |
| `Alt+Enter` | Open Ghostty |
| `Alt+C` | Close focused window |
| `Alt+1` ... `Alt+0` | Switch workspace |
| `Alt+Shift+1` ... `Alt+Shift+0` | Move window to workspace |
| `Alt+J/K/L/;` | Focus left/down/up/right |
| `Alt+Shift+J/K/L/;` | Move window left/down/up/right |
| `Alt+G` | Enter geometry mode |
| `Arrow` | Promote window to that screen half |
| `Shift+Arrow` | Resize focused pane |
| `B` | Balance panes in geometry mode |
| `Esc` / `Enter` | Exit geometry mode |
| `Alt+H/V` | Horizontal/vertical root tiles |
| `Alt+S/W/E` | Accordion/tiles layouts |
| `Alt+Space` | Search |
| `Alt+Tab` | App switcher |
| `Alt+P` | Power menu |
| `Alt+T` | Theme menu |
